实验九主要实现了调用函数完成三维图形几何变换。 
  运行main.cpp以后，按鼠标键盘，实现相应的三维几何变换，具体按键如下（切换为英文输入法）：
（1）按键”a“：表示向左平移0.1
（2）按键”s“：表示向下平移0.1
（3）按键”d“：表示向右平移0.1
（4）按键”q“：表示向外平移0.1
（5）按键”w“：表示向上平移0.1
（6）按键”e“：表示向里平移0.1
（7）按键”z“：表示放大
（8）按键”x“：表示缩小
（9）按键”u“：表示绕z轴逆时针旋转10度
（10）按键”o“：表示绕z轴顺时针旋转10度
（11）按键”i“：表示绕x轴逆时针旋转10度
（12）按键”k“：表示绕x轴顺时针旋转10度
（13）按键”j“：表示绕y轴逆时针旋转10度
（14）按键”l“：表示绕y轴顺时针旋转10度
